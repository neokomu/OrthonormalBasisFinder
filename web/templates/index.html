How To Use

Getting Started

This Orthonormal Basis Finder tool helps you find an orthonormal basis from a set of vectors using the Gram-Schmidt process.

Step-by-Step Instructions

1. Choose the number of Vectors: Use the + and − buttons to select how many vectors you want to input.

2. Set the Dimension of Vectors: Choose the dimension of your vectors (e.g., 2 for 2D, 3 for 3D).

3. Enter Your Vector Values: Input fields will appear based on your selection. Enter numerical values only.

4. Run the Computation: Click COMPUTE to apply the Gram-Schmidt orthogonalization.

5. View Results: The orthonormal basis will be displayed in the output section.

Example

If you entered two 2D vectors [3, 2] and [2, 1], this tool will compute an orthonormal basis using Gram-Schmidt process.
